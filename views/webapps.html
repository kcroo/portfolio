Trails REST API
Node.js/Express, Google App Engine, Datastore, Google People API, OAuth 2.0

This API serves data about trails, trailheads, and users using Google's NoSQL Datastore. Trails are created and "owned" by individual users, and each trail can be related to multiple trailheads. See GitHub for full documentation and usage. Note: as you'll see at the live website, frontend development was not the focus of this project!

Let's Go! Travel Planner
Flask (Python), MySQL, Bootstrap

This website allows users to plan their next vacation. They can add destinations to their trip and keep track of activities that they'll do at each destination. Authentication is handled by Flask-Login, with passwords encrypted in the MySQL database.
